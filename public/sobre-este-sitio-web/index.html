Este sitio fue diseñado y desarrollado sin el uso de plataformas prefabricadas o plantillas comerciales. Todo el código fue escrito manualmente en HTML, CSS y JavaScript, lo que permite un control total sobre el diseño, la estructura y el comportamiento visual del sitio.

El sistema de imágenes está gestionado a través de Cloudinary, asegurando una optimización automática en formato .webp, compresión inteligente y entrega rápida en múltiples dispositivos sin pérdida significativa de calidad.

Se utilizaron media queries personalizados para construir una versión móvil completamente independiente de la versión de escritorio. Esto significa que cada versión tiene sus propias proporciones, tamaños, márgenes y posicionamientos, garantizando que la experiencia visual esté adaptada específicamente a cada tipo de pantalla.

Además, se implementaron:

Animaciones CSS suaves para la carga de elementos visuales.

Un sistema de hover inteligente para escritorio, que desactiva enlaces hasta que el usuario interactúa visualmente con ellos.

Cada detalle visual, desde el margen de una imagen hasta el espaciado inferior del sitio, fue ajustado directamente en código para mantener la identidad gráfica del estudio. Esta página no es solo un portafolio, sino también una muestra práctica del enfoque técnico y visual de soto.
